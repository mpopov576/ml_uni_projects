**This project is based on a dataset provided by Restaurant GPT**

Restaurant ratings play a crucial role in determining visibility, customer acquisition, and revenue. Higher-rated restaurants tend to attract more users and generate greater business success.

The goal of this project is to analyze the factors that influence restaurant ratings—such as location, cuisine type, pricing, and availability of online services—and to build a machine learning model that predicts whether a restaurant's rating is above the median.

By understanding these factors, the project aims to support better recommendations for users and provide insights that can help new restaurants improve their performance.
